Ejercicio con Bootstrap

Contenedores

En HTML, uno puede ordenar su página de acuerdo al comando div. En bootstrap podemos aprovechar este elemento y agregarles una clase que nos facilitará el acomodamiento de estos elementos de forma mas lógica y visual.

Para acomodar los elementos usando bootstrap solo basta con agregar una clase, estas pueden ser de 3 distintos tipos.

container

La clase "container" nos delimita el área que tomará todo lo que contenga, dejando un márgen con le borde de la página.

container-fluid

La clase "container-fluid" nos delimita el área que tomará todo lo que contenga, pero usando todo el espacio disponible.

row

La clase "row" nos delimita las filas como de una tabla.

col

La clase "col" nos delimita las columnas como si de una tabla se tratase y nos permite agregar el contenido
